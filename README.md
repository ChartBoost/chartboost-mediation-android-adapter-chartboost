# Chartboost Mediation Chartboost Adapter

The Chartboost Mediation Chartboost adapter mediates Chartboost via the Chartboost Mediation SDK.

## Minimum Requirements

| Plugin                   | Version |
| ------------------------ |---------|
| Chartboost Mediation SDK | 5.0.0+  |
| Android API              | 21+     |

## Integration

In your `build.gradle`, add the following entry:
```
    implementation "com.chartboost:chartboost-mediation-adapter-chartboost:5.9.8.2.0"
```
